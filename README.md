# Celebal-Summer-Intern-Project
## Inspiration

I developed this fullstack chat application as it was assigned to me by the Celebal Technologies as a Summer Intern.
You can also try it.
## Technologies used

- React
- Next.js
- Node.js
- Express.js
- MongoDB
- Redux
- Socket.io
- TailwindCSS
- Cloudinary
- NodeMailer

## Features

- User authentication
- User profile
- upload picture
- User followers
- User notifications
- User chats
- User search

## Installation

1. Clone the repository
```bash
https://github.com/vaahu/Celebal-Summer-Intern-Project.git
```
2. Install the dependencies
```bash
npm install
```
3. Run the app in development mode
```bash
npm run dev
```
